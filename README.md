# Asteroids

A classic Asteroids arcade game clone built with Python and Pygame.

## Requirements

- Python 3.10+
- pygame 2.6.1

## Installation

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   ```

2. Activate the virtual environment:
   ```bash
   # macOS/Linux
   source venv/bin/activate

   # Windows
   venv\Scripts\activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Game

```bash
python main.py
```

## Controls

- **W** - Move forward
- **S** - Move backward
- **A** - Rotate left
- **D** - Rotate right
- **Space** - Shoot

## Gameplay

- Destroy asteroids by shooting them
- Avoid colliding with asteroids
- Larger asteroids split into smaller ones when hit
